# MOT Club 프로젝트 개요

## 프로젝트 소개

**MOT Club (Management of Technology Club)**은 한국산업기술진흥협회와 함께하는 기술경영 지식 공유 플랫폼입니다.

### 주요 목적
- 기술경영(MOT) 교육 수료생들의 지식 공유
- 기술경영에 관심 있는 사람들을 위한 커뮤니티
- 전문가와 일반 사용자 간의 상호작용 지원

## 기술 스택

### 프론트엔드
- **프레임워크**: Next.js 16.0.1 (App Router)
- **언어**: TypeScript 5.3.3
- **UI 라이브러리**: React 19.2.0
- **스타일링**: Tailwind CSS 3.4.0
- **상태 관리**: React Context API
- **주요 라이브러리**:
  - @headlessui/react - UI 컴포넌트
  - framer-motion - 애니메이션
  - axios - HTTP 클라이언트
  - react-quill - 리치 텍스트 에디터
  - react-pdf - PDF 뷰어

### 백엔드
- **서버**: Spring Boot (외부 연동)
- **API 통신**: RESTful API
- **포트**:
  - 개발: 8084
  - 프로덕션: 80 (www.motclub.co.kr)

### 인프라
- **컨테이너**: Docker
- **베이스 이미지**: Node.js 18 Alpine
- **빌드 모드**: Standalone (독립 실행 가능)

## 주요 기능

- 로그인/로그아웃
- 회원가입
- 비밀번호 재설정

- **학습 프로그램**: 교육 과정 관리

### 3. 관리자 기능
- 사용자 관리
- 전문가 관리
- 공지사항 관리
- 코드 관리
- 문의 관리
- 접근 로그 관리

### 4. 검색 기능
- 통합 검색
- 키워드 기반 검색
- 카테고리별 필터링

### 5. 파일 처리
- PDF 뷰어
- Excel 다운로드
- 파일 업로드/다운로드
- 이미지 처리

## 프로젝트 구조

```
mot_prj/
├── src/
│   ├── app/              # Next.js App Router 페이지
│   ├── components/       # React 컴포넌트
│   ├── context/          # React Context (인증)
│   ├── config/           # API 설정
│   ├── hooks/            # 커스텀 훅
│   ├── utils/            # 유틸리티 함수
│   └── types/            # TypeScript 타입 정의
├── public/               # 정적 파일
├── docs/                 # 프로젝트 문서
├── Dockerfile            # Docker 이미지 빌드
└── [설정 파일들]
```

## 빌드 및 실행

### 개발 환경
```bash
npm run dev              # 로컬 개발 서버
npm run dev:dev          # 개발 서버 연동
npm run dev:staging      # 스테이징 서버 연동
```

### 프로덕션 빌드
```bash
npm run build:prod       # 프로덕션 빌드
npm run copy-public      # public 폴더 복사
npm run startHub         # 프로덕션 서버 실행
```

### Docker
```bash
docker build -t mot-club .
docker run -p 3000:3000 mot-club
```
